# 青海大学附属医院职称评审与晋升管理办法

## 1. 总则
为激励专业技术人员提升业务水平，规范职称评审工作，结合青海省卫生健康委及青海大学相关要求，特制定本办法。

## 2. 评审序列与层级
* **医疗类**：住院医师 -> 主治医师 -> 副主任医师 -> 主任医师
* **护理类**：护士 -> 护师 -> 主管护师 -> 副主任护师 -> 主任护师
* **教学/科研类**：助教/研习员 -> 讲师/助理研究员 -> 副教授/副研究员 -> 教授/研究员

## 3. 核心晋升条件（以副高级为例）
1. **学历与资历**：取得中级职称后，受聘该岗位满规定年限（通常为5年，博士学历可缩短）。
2. **临床能力**：完成规定的门诊量、管床数、手术台次（按手术级别划分），且医疗质量考核合格。
3. **科研与教学**：
   * 参与或主持省级及以上科研项目。
   * 在核心期刊发表专业学术论文。
   * 承担青海大学临床医学院的本科生、研究生带教任务。
4. **基层支援经历（硬性指标）**：
   * 晋升高级职称前，必须完成至少半年的对口支援基层医疗卫生机构（如玉树、果洛、海西等州县级医院）的工作任务，且考核合格。
5. **继续医学教育（CME）**：每年必须修满规定的国家级和省级继续医学教育学分。

## 4. 破格晋升
在重大突发公共卫生事件（如抗击疫情、抗震救灾）中表现突出，或取得重大科研突破者，可按青海省相关政策申请破格晋升。
